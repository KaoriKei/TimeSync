# 予実管理アプリ仕様書

## 📱 アプリ概要
**アプリ名**: TimeSync（仮）  
**目的**: 予定時間と実際時間の簡単な記録・比較で時間管理を改善  
**ターゲット**: 時間管理に悩む個人ユーザー  
**プラットフォーム**: iOS（Apple Store申請予定）

## 🎯 コンセプト
- **超シンプル**: スケジュール帳の左右分割のような直感的UI
- **クイックアクセス**: ウィジェットから0タップで記録開始
- **習慣化支援**: 定期通知で記録忘れを防止
- **予実比較**: 「30分のつもりが20分で終わった」を可視化

## 🚀 MVP機能（週末実装目標）

### 1. 基本画面
#### メイン画面
```
┌─────────┬─────────┐
│  予定   │  実際   │
├─────────┼─────────┤
│ 📝 資料作成│ 📝 資料作成│
│ 30分    │ 25分    │
├─────────┼─────────┤
│ ☕ 休憩  │ ☕ 休憩  │
│ 15分    │ 20分    │
└─────────┴─────────┘
```

- 左：予定（事前入力）
- 右：実際（リアルタイム記録）
- 1画面で予実比較が一目瞭然

#### 入力画面
- **予定入力**: タスク名 + 予定時間
- **実際記録**: タイマー開始/停止 または 手動入力
- **カテゴリ**: 仕事、休憩、移動、その他（アイコン付き）

### 2. ウィジェット機能
#### 小サイズウィジェット
```
┌─────────────────┐
│ 📝 今の行動記録   │
│ [タップして入力]   │
│ 今日: 3.5h記録済  │
└─────────────────┘
```

#### 中サイズウィジェット
```
┌─────────────────┐
│ 仕事 | 休憩 | 移動 │
│ 📊 | ☕ | 🚶   │
│ 予定: 6h | 実際: 5.5h │
└─────────────────┘
```

### 3. 通知機能
#### 定期リマインダー
- **間隔設定**: 15分、30分、1時間、2時間、カスタム
- **時間帯設定**: 平日のみ、休日のみ、カスタム時間帯
- **通知文**: 「今何してる？📝 予定と実際を記録しよう」

#### スマート通知
- 予定時間到達時: 「予定時間になりました！」
- 予定時間超過時: 「予定より5分オーバーしてます」

### 4. 課金要素（ハッカソン要件）
#### 無料版
- 基本的な記録機能
- ウィジェット（小サイズのみ）
- 1日分のデータ表示

#### Pro版（月額）
- **詳細分析**: 週次・月次レポート
- **高度なウィジェット**: 中・大サイズ
- **データエクスポート**: CSV出力
- **カスタマイズ**: テーマ、アイコン変更
- **無制限履歴**: 過去データ無制限保存

## 🛠 技術仕様

### 開発環境
- **言語**: Swift
- **フレームワーク**: SwiftUI
- **IDE**: Xcode
- **開発支援**: Cursor + Claude

### 主要技術
- **ウィジェット**: WidgetKit
- **通知**: UNUserNotificationCenter
- **データ保存**: Core Data（ローカル）
- **UI**: SwiftUI + Combine

### データ構造
```swift
struct TimeEntry {
    let id: UUID
    let taskName: String
    let category: TaskCategory
    let plannedDuration: TimeInterval
    let actualDuration: TimeInterval?
    let startTime: Date
    let endTime: Date?
    let isCompleted: Bool
}

enum TaskCategory {
    case work, break, commute, other
}
```
